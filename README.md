# 🛒 Sistema de Carrito de Compras en Java

## 📖 Descripción
Aplicación en Java que permite crear productos, agregarlos a un carrito y calcular el total de la compra.

## 🧠 Conceptos aplicados
- Programación Orientada a Objetos (POO)
- Encapsulamiento
- Uso de clases y objetos
- Constructores
- Getters y Setters
- ArrayList

## 📂 Estructura
- Producto.java: Define nombre, precio y cantidad del producto
- Carrito.java: Maneja la lista de productos y calcula el total
- App.java: Contiene el método main y la ejecución del programa
